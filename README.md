# Proyecto Boda: Landing Page

¡Bienvenidos al repositorio inicial del proyecto web de la Boda de David y Rocío!

## Características
* Creado usando **Next.js 15**, **React** y el versátil framework **Tailwind CSS v4**.
* Cronómetro regresivo completamente en sincronización UTC-3 para el Gran Día: **3 de Abril, 2027**.
* Interfaz con componentes adaptativos y modo Dark Glassmorphism.
* Renderización y validaciones ajustadas con Linters modernos.

*Nota: Actualmente el proyecto opera el desarrollo de la Fase 1. Una Fase 2 incluirá un Panel Administrativo y un Área de Usuarios conectada a una base de datos.*
